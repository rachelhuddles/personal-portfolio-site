Altar'd State

Database and Web Application

A complete overhaul of Altar'd State's Distribution Center data entry, storage, and reporting processes. I began by cleaning and converting the data stored across over 32,400 excel worksheets to a SQL Server Database. Next, I designed and built a 38-page internal company website using ASP.NET, SQL, VB, HTML, CSS, JavaScript, and Bootstrap which allows over 40 users to interact with the database through a variety of tools and reports. I then built out new reporting features which have significantly reduced the time needed to complete daily tasks in several departments. Since launching this database and web application, productivity, efficiency, and accuracy in reporting have significantly increased.
